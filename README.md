## Usage 🧰

**When `emojiFormat` is set to `code`, some gitmoji may not be displayed properly on Github.**

**Please set `emojiFormat` to `emoji`.**

```json
{
  "emojiFormat": "emoji",
  "presets": ["@sno2wman/gitmoji-preset"]
}
```
